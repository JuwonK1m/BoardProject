# Board (게시판 프로젝트)
## 기능
- 회원가입
- 로그인/로그아웃
- 게시물 CRUD
- 게시물 검색(제목, 내용, 글쓴이, 댓글 내용, 댓글 작성자, 통합)
- 조회수
- 댓글
- 게시물 정렬(최신순, 조회순, 댓글순)
- 내 글 보기
- 추천/비추천
- 즐겨찾는 게시물
- 페이지네이션
- Todo 리스트
## 개발 기간
- 2020.03.30 ~ 2020.04.19
## 회고
- 인생 첫 Spring 프로젝트라 맨땅에 헤딩하는 식으로 개발하였더니 미숙한 점이 많음
- Back-End 개발자를 희망하는 만큼, view 코드보다 api 설계에 더욱더 집중할 것
- api 코드의 내부 성능 향상을 위해 많이 고민해보아야 함
